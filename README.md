# Cloud-Computing-Documentation
This is a collection of my AWS and Google cloud work, including beginner friendly guides of resources I have provisioned.

All of the content featured here was worked on during my time at governors state in a cloud computing class

All work was well documented and intended for someone who was new to cloud computing to follow along

The work will switch between google cloud and AWS, as I was learning how to navigate and troubleshoot both

All files include:

1. Screenshots of anything I did
2. explinations of the step to provision the resource
3. A detailed explination as to why I did this and what it solves
4. (some) have reflection questions and answers
5. Troubleshooting notes of real issues I ran into and solved
